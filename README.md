# 🏛️ CityUHK 学术导航

**香港城市大学 · 学院 → 学系 → 教授 → 课题组**

一个单一 HTML 文件的学术导航网站，帮助申请者快速了解 CityU 各院系导师的研究方向与课题组情况。

## 📊 数据概况

- **799 位教授** — 覆盖 10 大学院、24 个学系
- **全部个性化研究兴趣** — 每位教授 3 个中文研究方向，告别模板化描述
- **全部中文姓名** — 含音译名
- **344 名博士生** — 分布在 125 位导师名下，含论文题目
- **数据来源** — 各学院官网 + [CityU Scholars](https://scholars.cityu.edu.hk) 公开论文记录

## 🚀 使用方式

直接打开 `index.html` 即可，无需服务器。或访问：

**[yifeilu84-sketch.github.io/cityuhk-academic](https://yifeilu84-sketch.github.io/cityuhk-academic/)**

## ⚠️ 声明

- 本网站仅用于公益目的，旨在消除学术信息不对称
- 部分学系未明确列出博士生信息或未注明导师，因此未全部收录
- 具体信息以 [cityu.edu.hk](https://www.cityu.edu.hk) 官方公布为准
- 祝大家都能找到适合自己的学术道路 🎓
