# 🏛️ CityUHK 学术导航

**香港城市大学 · 学院 → 学系 → 教授 → 课题组**

一个单一 HTML 文件的学术导航网站，帮助申请者快速了解 CityU 各院系导师的研究方向与课题组情况。

🔗 **在线访问：[yifeilu84-sketch.github.io/cityuhk-academic](https://yifeilu84-sketch.github.io/cityuhk-academic/)**

## 📊 数据概况

| 类别 | 数量 |
|------|------|
| 🏛️ 学院 | 10 |
| 🏢 学系 | 31 |
| 👨‍🏫 教授 | 928 |
| 🎓 博士生 | 565（仅统计官网公开名单，非全部在读） |
| 📄 论文 | 2,604 篇（1,149 篇含有效 DOI 链接） |

- **全部个性化研究兴趣** — 每位教授 3 个中文研究方向，告别模板化描述
- **全部中文姓名** — 含音译名
- **教授个人主页链接** — 官网 / Google Scholar / ORCID
- **高被引论文** — 来自 Crossref / CityU Scholars 公开记录
- **数据来源** — 各学院官网 + [CityU Scholars](https://scholars.cityu.edu.hk)

## 🚀 使用方式

直接打开 `index.html` 即可浏览全部数据，无需服务器。或通过 GitHub Pages 在线访问。

### 导航说明

1. 左侧栏选择 **学院**，查看下属学系及教授人数
2. 点击 **学系卡片** 进入教授列表，可按研究方向筛选
3. 点击 **教授卡片** 展开个人主页、谷歌学术、高被引论文
4. 顶部 **搜索框** 支持教授姓名、研究方向、学院/学系模糊搜索
5. 手机端自动适配，左上角 ☰ 展开菜单
